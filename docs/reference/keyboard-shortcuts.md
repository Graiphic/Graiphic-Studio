# Keyboard Shortcuts

## Documents

| Shortcut | Action |
| --- | --- |
| `Ctrl+N` | New document |
| `Ctrl+O` | Open document |
| `Ctrl+S` | Save |
| `Ctrl+Shift+S` | Save As |
| `Ctrl+W` | Close |
| `Ctrl+Q` | Exit |

## Editing

| Shortcut | Action |
| --- | --- |
| `Ctrl+Z` | Undo |
| `Ctrl+Shift+Z` | Redo |
| `Ctrl+C` | Copy selected content |
| `Ctrl+V` | Paste |
| `Delete` or `Backspace` | Delete selection |
| Arrow keys | Nudge selection by one grid unit |
| `Shift` + arrow | Nudge by a larger step |
| `Escape` | Cancel the active mode or tool |

## Pointer Modifiers

| Modifier | Action |
| --- | --- |
| `Shift` while moving | Lock to the first horizontal or vertical axis |
| `Ctrl` while moving | Drag a duplicate while held |
| `Shift` while resizing | Preserve proportions |
| `Shift` while selecting | Add or remove objects from a multi-selection |
| Middle mouse drag | Pan the Front Panel or Icon Editor |

## Text And Icon Editing

| Shortcut | Action |
| --- | --- |
| `Enter` | Validate label editing |
| `Shift+Enter` | Insert a label line break |
| `Ctrl+Shift+Delete` | Clear all Icon Editor layers |

Some commands are context-sensitive. A shortcut has no effect when the current
selection cannot perform the operation.

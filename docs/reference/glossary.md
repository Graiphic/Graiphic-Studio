# Glossary

## Graiphic Studio

The public product name for the visual IDE.

## .frog

The source document format. It owns widget instances, layout, labels, bindings,
initial values, icon data, and instance-level overrides.

## Front Panel

The visual editing surface where user-facing widgets and labels are arranged.

## Widget

A reusable visual element placed on the Front Panel.

## Label

The text label associated with a widget or a standalone text element. A label is
not the same as the widget value text.

## Aura

The selection or placement outline around an object.

## Interface Map

The public interface square that shows the visual arrangement of public input
and output slots.

## Interface Layout Pattern

The selected slot distribution pattern used by the Interface Map.

## Front Panel Binding

The explicit source-level association between a widget value and a public
interface slot or port.

## Icon Editor

The built-in editor for the document icon.

## Runtime

The execution side that consumes explicit `.frog` source artifacts. Runtime
behavior should not depend on hidden Studio-only state.

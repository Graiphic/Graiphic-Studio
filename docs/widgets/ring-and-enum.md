# Ring And Enum Widgets

Ring and Enum widgets represent selectable values.

They are grouped together in the Widget Navigator because both are list-like
selection widgets.

## Ring

A Ring widget displays a selected item from a list.

The Ring context menu supports editing items. The item editor allows users to:

- insert items
- delete items
- move items up or down
- disable items
- edit labels directly in the item list
